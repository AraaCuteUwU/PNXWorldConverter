### Usage

`java -jar pnxworldconvert.jar -t save_path -d dimension_name`

### Params

**save_path example:**

```
D:\your device path\1.19.2\.MINECRAFT\SAVES\new world  ←this is save_path
├─advancements
├─data
├─datapacks
├─DIM-1
│  ├─data
│  ├─entities
│  └─region
├─DIM1
│  └─data
├─entities
├─playerdata
├─poi
├─region
└─stats
```

**Dimension_name valid values: `OVERWORLD`  `NETHER` `END`**
